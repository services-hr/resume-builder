# Google ドキュメント テンプレート設計仕様

このドキュメントは、Googleドキュメント上で**実際に手作業で作成する**職務経歴書テンプレートの設計仕様です。以下の手順どおりにGoogleドキュメントを作成してください。

---

## 作成手順（およそ15〜20分）

### ステップ1：新規ドキュメントを作成

1. https://docs.google.com にアクセス
2. 「空白のドキュメント」をクリック
3. ファイル名を **「職務経歴書テンプレート」** に変更

### ステップ2：ページ設定

上部メニューから：
- **ファイル → ページ設定**
  - ページサイズ：**A4**
  - 向き：**縦**
  - 余白（上下左右）：**20mm**（デフォルトの25.4mmより狭く）
- **表示 → 印刷レイアウト** にチェックが入っていること

### ステップ3：全体のフォントとサイズ設定

- フォント：**Noto Sans JP**（候補に出ない場合は「フォントをさらに表示」から追加）
- 基本フォントサイズ：**10.5pt**

### ステップ4：本文を以下の構成で作成

---

## ▼ ここから下がテンプレートの中身 ▼

以下のコンテンツをそのままGoogleドキュメントに貼り付けてください。`{{xxx}}`のプレースホルダはそのまま残します（GASがここを実データに置換します）。

---

### タイトル部分（中央揃え）

```
職 務 経 歴 書
```

- フォントサイズ：**20pt**
- 太字：**はい**
- 色：**#2D5A45**（深緑）
- 文字間：**8pt空ける**（各文字の間にスペース）
- 段落の下に **2pt の下線**（色：#2D5A45）

次の行（右揃え）：

```
{{today}} 現在
```

- フォントサイズ：**9pt**
- 色：**#6B6B6B**（グレー）

空行を1行入れる。

---

### 基本情報セクション（2列の表）

**挿入 → 表 → 2×2** の表を挿入し、以下のように設定：

| ヘッダー（左列、背景色 #F5F3F0） | 内容（右列） |
|---|---|
| **氏名** | {{lastNameKana}} {{firstNameKana}}<br>**{{lastName}} {{firstName}}**　　{{birthDate}}生 （{{age}}） |
| **連絡先** | {{address}}<br>TEL: {{phone}}　　Email: {{email}}<br>最寄駅: {{nearestStation}} |

💡 ヒント：`{{fullName}}` を使うと「山田 太郎」のようにスペース区切りで姓名を結合した文字列になります。`{{fullNameKana}}` も同様。レイアウト都合でどちらを使ってもOKです。

表の設定：
- 左列の幅：**30mm**、右列の幅：**残り**
- 左列：背景色 `#F5F3F0`、太字、フォントサイズ 10pt
- 右列：
  - フリガナの行：フォントサイズ 9pt、色 `#6B6B6B`
  - 氏名（`{{lastName}} {{firstName}}`）の部分：フォントサイズ 14pt、太字
  - `{{birthDate}}生 （{{age}}）` の部分：フォントサイズ 10pt、色 `#6B6B6B`
  - `{{address}}`、`TEL`、`Email`、`最寄駅`：フォントサイズ 10pt
- 枠線：全辺 0.5pt、色 `#E2DDD7`

---

### 職務経歴セクション（見出し＋展開ブロック）

見出し：

```
■ 職務経歴
```

- フォントサイズ：**12pt**
- 太字：**はい**
- 色：**#2D5A45**
- 段落の下に **0.5pt の下線**（色：#E2DDD7）
- 上マージン：**16pt**、下マージン：**8pt**

次の段落にプレースホルダ：

```
{{careers}}
```

- この段落は **そのまま1行で残す** こと（GASがこの位置に職務経歴の繰り返しブロックを挿入します）
- フォントサイズ：10.5pt
- **重要**：この `{{careers}}` 段落を含む段落スタイルは「標準テキスト」にしておいてください。見出しスタイルになっていると置換後の書式が崩れます

---

### スキルセクション

見出し：

```
■ スキル
```

- 上と同じ書式（12pt、太字、#2D5A45、下線）

本文プレースホルダ：

```
{{skills}}
```

- フォントサイズ：10.5pt
- 背景色：**#FAFAF8**（薄いグレー）
- 段落のパディング：上下左右 **4pt**
- `\n` は改行として解釈されます

---

### 保有資格・免許セクション

見出し：

```
■ 保有資格・免許
```

本文プレースホルダ：

```
{{qualifications}}
```

- フォントサイズ：10.5pt
- ※ GAS側で `日付\t資格名` の形式の文字列を挿入するため、タブストップを30mmに設定しておくと日付と資格名がきれいに揃います

---

### 自己PRセクション

見出し：

```
■ 自己PR
```

本文プレースホルダ：

```
{{pr}}
```

- フォントサイズ：10.5pt
- 背景色：**#FAFAF8**
- 行間：**1.8倍**

---

以上

- フォントサイズ：9pt
- 色：#6B6B6B
- 右揃え
- 上マージン：24pt

## ステップ5：テンプレートのドキュメントIDを取得

1. 作成したドキュメントのURLを確認します
2. URLは `https://docs.google.com/document/d/**XXXXXXXXXX**/edit` という形式
3. **XXXXXXXXXX** の部分がドキュメントIDです
4. このIDを後でGASのスクリプトプロパティ `TEMPLATE_DOC_ID` に設定します

---

## プレースホルダ一覧（チェックリスト）

テンプレートを作ったら、以下のプレースホルダがすべて含まれていることを確認してください：

- [ ] `{{today}}` — 本日の日付
- [ ] `{{lastName}}` — 姓
- [ ] `{{firstName}}` — 名
- [ ] `{{lastNameKana}}` — 姓フリガナ
- [ ] `{{firstNameKana}}` — 名フリガナ
- [ ] `{{birthDate}}` — 生年月日
- [ ] `{{age}}` — 年齢
- [ ] `{{email}}` — メールアドレス
- [ ] `{{phone}}` — 電話番号
- [ ] `{{address}}` — 住所
- [ ] `{{nearestStation}}` — 最寄駅
- [ ] `{{careers}}` — 職務経歴（展開ブロック）
- [ ] `{{skills}}` — スキル
- [ ] `{{qualifications}}` — 資格・免許
- [ ] `{{pr}}` — 自己PR

**追加で使えるオプション：**
- `{{fullName}}` — 「山田 太郎」のように姓名を半角スペースで結合
- `{{fullNameKana}}` — 「ヤマダ タロウ」のようにフリガナを結合

## 注意事項

### 1. プレースホルダの書き方

**必ず半角の中括弧2つ** `{{` と `}}` で囲みます。全角の `｛｛｝｝` では置換されません。

### 2. プレースホルダの間に書式が混ざらないこと

たとえば `{{name}}` の途中で太字が切れていると、Googleドキュメント内部では `{{name}}` ではなく別のテキストとして扱われ、置換が失敗します。プレースホルダ全体に同じ書式を適用してください。

### 3. 書式のテストにはサンプルデータを使う

GASエディタから `testGenerate` 関数を実行すると、サンプルデータでPDFが生成されます。出来上がりを確認して、崩れがあればテンプレートを調整してください。

### 4. プレビューUI と 100% 一致はしない

Googleドキュメントの制約上、CSSのような細かい装飾は再現できません。「プレビューに似た雰囲気」までは持っていけますが、たとえば：

- 角丸の背景（`border-radius`）
- 複雑な影（`box-shadow`）
- 一部の字間調整

これらは再現できません。PDFとして成立するきれいな日本のビジネス文書スタイルが目標です。
